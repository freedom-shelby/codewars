# Even or Odd

**Difficulty:** 8 kyu  
**Language:** Go  
**Link:** https://www.codewars.com/kata/53da3dbb4a5168369a0000fe/go

## Description

Create a function that takes an integer as an argument and returns "Even" for even numbers or "Odd" for odd numbers.

## Topics
- Mathematics  
- Fundamentals

## Notes

- Input is always an integer.
- Output must be exactly `"Even"` or `"Odd"`.
- Simple modulus operation: `n % 2 == 0`.
